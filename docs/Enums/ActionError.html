ActionError

public enum ActionError<Error: Swift.Error>: Swift.Error

The type of error that can occur from Action.apply, where Error is the type of error that can be generated by the specific Action instance.

  • disabled

    The producer returned from apply() was started while the Action was disabled.

    Declaration

    Swift

    case disabled
  • producerFailed

    The producer returned from apply() sent the given error.

    Declaration

    Swift

    case producerFailed(Error)
